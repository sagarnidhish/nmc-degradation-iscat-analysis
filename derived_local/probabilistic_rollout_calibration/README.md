# Probabilistic Rollout Calibration

Audits empirical conformal-style residual bands for particle-ROI rollout baselines.

The analysis leaves one event-reference cycle out, estimates MAE quantiles from the remaining ROI-frame residuals, and checks coverage in all, near-transition, far-from-transition, event, and control strata.

Use this as uncertainty and review-prioritization evidence, not as a final probabilistic video model or calibrated transport estimate.
