# Masked Residual Transition Timing

Aligns masked particle-rollout residual timing with automatic phase-kinetic transition times.

- ROI-method rows: 156
- ROI count: 52
- Permutations per alignment null: 5000

Outputs:

- `masked_residual_transition_timing_per_roi.csv`: per ROI-method timing features.
- `masked_residual_transition_timing_alignment_tests.csv`: peak/weighted-center transition alignment nulls.
- `masked_residual_transition_timing_event_control_tests.csv`: event/control timing contrasts.
- `masked_residual_transition_timing_correlations.csv`: timing links to kinetic descriptors.
- `masked_residual_transition_timing_ranked.csv`: ROI-method rows ranked by near-transition residual concentration.
