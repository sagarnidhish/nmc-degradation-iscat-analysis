## Guardrail

Pre-event physics modes are unsupervised clusters of automatic source-residual ROI features. Mode enrichment and event-clock tests nominate repeatable front/diffusion/heterogeneity states for review; they are not manual degradation labels, calibrated phase boundaries, diffusion coefficients, or causal forecasts.
